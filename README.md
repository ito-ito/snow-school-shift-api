# snow-school-shift-api

## Setup

```sh
git clone https://github.com/ito-ito/snow-school-shift-api.git
cd snow-school-shift-api
```

環境変数をコピーし、各項目を設定します

```sh
cp ./.env.sample ./.env
cp ./src/env.sample.ts ./src/env.ts
```

## Push Google App Script

```sh
npm run push
```

## Open Google App Script Editor

```sh
npm run open
```
